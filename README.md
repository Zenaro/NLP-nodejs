# 自然语言处理
- 项目使用express为框架，pug作为前端渲染引擎

- 若尚未安装nodejs环境，可前往[官网](https://nodejs.org/en/)下载

## 脚本命令
初始化项目：
### `npm install`

完成工具安装后，可在项目下执行:

### `npm start`
